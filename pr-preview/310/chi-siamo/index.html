DataCivicLab è uno spazio civico aperto dove proviamo a rendere i dati pubblici più leggibili, utili e condivisibili.

Come funziona

Il Lab segue un percorso lineare dalla domanda all'output:

  1. Domanda civica — chiunque può aprire una Discussion
  2. Scouting — verifichiamo se esistono fonti pubbliche per rispondere
  3. Incubazione — costruiamo pipeline dati riproducibili
  4. Analisi — produciamo notebook e finding pubblici
  5. Catalogo — i dataset puliti finiscono in data-explorer

Principi

  • Lavoriamo al ritmo che i dati permettono, non più veloce
  • Tutto è pubblico per default
  • Non servono competenze tecniche per partecipare
  • I dati non parlano da soli — servono domande, metodo e contesto

Community

  • GitHub Discussions — proposte, domande, confronto
  • Discord — scambio veloce
  • LinkedIn — aggiornamenti pubblici
  • Open Board — lavoro in corso
